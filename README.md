# wued-dreamer

A fullstack deved for my dreamer

## 💻 FrontEnd & Client

考虑到团队整体水平，我们决定采用 `vue` 进行开发。

### 本地开发

`cd client`

```npm
npm run dev
```

`http://localhost:3000/`

### 生产构建

`cd client`

```npm
npm run build

npm run start
```

## 🖥 Backend & Server

本地开发数据库 `mongodb://127.0.0.1:27017`
